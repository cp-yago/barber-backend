## How to execute this project

```bash
# clone this repo

$ git clone https://github.com/cp-yago/barber-backend.git

# or ssh
$ git clone git@github.com:cp-yago/barber-backend.git

# install dependencies

$ yarn

# Run docker compose
$ docker-compose up -d

# Run migrations

$ yarn typeorm migration:run

# Run project

$ yarn dev:server
```
